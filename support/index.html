FAQ

  • Supported devices: iPhone running iOS 17.0 or later
  • No account or sign in is required
  • All app data is stored locally on-device
  • Camera access is optional and only used for item and entry photos
  • Catalog import from URL is optional

Sample Catalog Import URL

You can test catalog import with https://integrationuser92.github.io/goodc-assets/catalog-index.json.

Delete Data

To remove your local data, open Settings in the app and choose Delete All Data.
